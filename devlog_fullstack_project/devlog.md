## 개인 블로그
- devlog 메인 화면에서 블로그 없을 시 생성 버튼을 통해 블로그 생성
- 블로그명, 주요 카테고리 설정(IT, 패션, 등등)
- 개인 블로그 설정 메뉴 필요(블로그 명 변경, 개인 블로그 내 폴더 생성, 테마 변경(생각중), 화면 배치(생각중))
